This project has been created as part of my HNC Electrical and Electronic Engineering assignment.
The task set was to design an IoT device that
1. Collects data through sensors (temperature, light levels and at least one more type of environmental information)
2. Publishes data to a server using wi-fi and MQTT
3. Compares data to thresholds and responds through activators accordingly
4. Has an online dashboard that displays data and through which thresholds can be edited

It also needs to:
- Be reliable and testable.
- Be written primarily in micro-python
- Use a Raspberry Pi Pico 2W

I have worked through the following sequence:
- Code 1. Measures temperature, compares it to a pre-set threshold, and illuminates an LED accordingly.
- Code 2. Adds in a light level sensor and LED, plus a humidity sensor and LED.
- code 3. Adds in wifi and MQTT
- code 4. Plays with where the logic is hosted (Pico or FlowFuse), adds a Heater Boost button and adds an Adafruit INA228 to monitor power generated by a small solar panel.
